### ** Skills Page (skills.html)**

## How to Run the Project  

1. Download or clone the project folder.  
2. Keep all `.html` files and `style.css` in the same directory.  
3. Open `index.html` in any web browser (Chrome, Edge, Firefox).  
4. Navigation links will take you to other pages.
